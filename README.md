## Installation

1. Clone git repository,

``` bash 
git clone https://github.com/derrickais/e-retailr.git
```

2. Run npm install

``` bash 
npm i
```

3. Add .env file and add your SQL login info
``` bash 
touch .env
```

## Usage

Open Insomnia and use the routes given in the routes file
